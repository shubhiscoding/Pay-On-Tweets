# Steps To Setup on Local Host:

### Step 1: ``` Clone the repo ```
### Step 2: ```Navigate to the repo's directory in the terminal and run:```
```
npm install
```
### Step 3:
```
npm start
```

### Step 4: ```Navigate to [ http://localhost:3000/ ]```
